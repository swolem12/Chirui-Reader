## How to Use

1. Click the link above
2. Click the green **"Run workflow"** button (top right)
3. Enter the version tag (e.g., `v0.1.0`)
4. Click **"Run workflow"** to start the build
5. Wait 5-10 minutes for the workflow to complete
6. Check [Releases](https://github.com/swolem12/Chirui-Reader/releases) for your new release with APK files
